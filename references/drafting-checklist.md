# 起草检查清单 v1.0

**适用范围：** legal-review 起草模式生成的合同草稿，在交由法律审查前的自检清单。

**适用立场：** 投资方 / 总包方（不同立场，重点关注点不同）

---

## 一、当事人信息完整性与真实性

| 检查项 | 投资方重点 | 总包方重点 |
|--------|-----------|-----------|
| 甲乙方名称与营业执照一致 | ✅ 甲方（投资方）信息准确 | ✅ 乙方（承包方）资质证照齐全 |
| 注册地址与实际经营地址一致 | ✅ 避免送达风险 | ✅ 确认项目所在地法院管辖 |
| 法定代表人姓名与身份证一致 | ✅ | ✅ |
| 联系人及联系方式明确 | ✅ | ✅ |
| 签约代表授权文件（如有） | — | ✅ 总包方需确认有完整授权 |

---

## 二、核心条款完整性

| 条款类型 | 必须包含 | 投资方视角 | 总包方视角 |
|---------|---------|-----------|-----------|
| **标的条款** | 工程/项目名称、范围、内容 | 资产边界清晰，无歧义 | 工程范围明确，避免范围不清导致亏损 |
| **价款条款** | 金额、计价方式、支付节点 | 付款节奏可控，验收标准明确 | 付款节点合理，预付款覆盖前期成本 |
| **工期条款** | 开工/竣工日期、节点工期、延期责任 | 工期延误违约责任明确 | 工期顺延条件充分（设计变更/甲方付款延迟等） |
| **变更条款** | 变更范围、变更流程、变更计价 | 变更定价机制合理 | 变更定价有据可依，不被压价 |
| **质量条款** | 验收标准、保修期、质保金 | 质保金退还条件清晰 | 质保金比例合理（通常5-10%） |
| **违约条款** | 违约金比例、赔偿范围 | 我方（甲方）违约成本低 | 对方（甲方）逾期付款违约成本充分 |
| **解除条款** | 解除条件、解除程序 | 我方单方解除权保留 | 甲方欠款时停工权保留 |
| **争议条款** | 争议解决方式、管辖约定 | 管辖法院对我方便利 | 管辖法院对总包方便利（通常为项目所在地） |

---

## 三、双立场必检项

### 投资方必检（甲方立场）

- [ ] 资产/股权交割条件和时间节点明确
- [ ] 股转款支付与股权过户绑定（付款=过户同时完成）
- [ ] 陈述与保证条款中对方（乙方/转让方）保证范围充分
- [ ] 违约情形覆盖了核心风险（施工质量/股权瑕疵/陈述失实）
- [ ] 我方单方解除权保留（在重大风险出现时可止损）
- [ ] 争议解决约定我方所在地或对我方便利的法院

### 总包方必检（乙方立场）

- [ ] 预付款比例足以覆盖前期投入（通常不低于合同价20%）
- [ ] 工期顺延条件明确列举（设计变更、甲方延迟付款、不可抗力等）
- [ ] 变更计价机制：有合同单价则按单价，无单价则按实际成本+合理利润
- [ ] 进度款逾期支付：明确约定承包人有权暂停施工且不算违约
- [ ] 质保金退还时限明确（通常为缺陷责任期满后30日内）
- [ ] 承包人欠款时停工权的保留条款

---

## 四、法律合规必检

- [ ] 合同效力条款：无违反法律法规强制性规定
- [ ] 格式条款（如有）：已尽合理提示义务
- [ ] 保密条款：双方均有保密义务（双向保密优先）
- [ ] 知识产权条款：各方已有IP归属清晰，新增IP共有机制明确
- [ ] 不可抗力条款：覆盖疫情、政策变化等，通知义务和免责条件明确
- [ ] 适用法律明确为中华人民共和国法律

---

## 五、占位符与待确认项

- [ ] 所有待填内容均用 `[待填写：xxx]` 标注，无遗漏
- [ ] 金额/日期等数值类占位符已标注单位（如「人民币[待填写：金额]元」）
- [ ] 甲乙方名称、地址、法定代表人等主体信息已填写完整（非占位符）
- [ ] 签约日期已填写或标注「签署当日」

---

## 六、可选增强条款（按需添加）

| 条款 | 说明 | 建议方 |
|------|------|--------|
| 优先续约权 | 协议期满后同等条件下优先续约 | 投资方 |
| 竟业限制 | 关键人员在职/离职后竞业限制 | 总包方 |
| 情势变更 | 合同订立后客观情况剧变可重新协商 | 双方 |
| 环保合规 | 项目建设符合环保法规要求 | 投资方 |
| 反腐败条款 | 双方承诺不行贿、不收受贿赂 | 双方 |

---

## 七、生成后动作

1. **自检**：按本清单逐项勾选，确保无明显遗漏
2. **立场检查**：确认合同整体保护我方核心利益
3. **提交审查**：调用 legal-review 主流程进行三维法律分析
4. **输出**：起草检查清单勾选结果作为背景信息一并提交
