# 合同类型法律分析参考手册

本文档为法律审查技能提供各类典型合同的专项分析要点，覆盖合同核心要素、常见争议条款、各方核心关切。适用于合同审查、条款谈判、风险识别等场景。

---

## 1. EPC/工程总包合同（设计-采购-施工一体化）

### 1.1 Overview and Core Elements

**合同性质：** EPC（Engineering, Procurement, Construction）合同即设计、采购、施工一体化总承包合同，是国际及国内大型工程建设项目中广泛采用的合同模式。其核心特征是将工程的设计、采购、施工三大环节整体打包交由一家承包商负责，业主侧重宏观管理而非具体实施。

**主要法律依据：**
- 《中华人民共和国民法典》合同编（第三编 合同关系）
- 《建设工程施工合同（示范文本）》（GF-2017-0201）
- 《建设项目工程总承包合同（示范文本）》（GF-2020-0216）
- FIDIC银皮书（Conditions of Contract for EPC/Turnkey Projects）

**核心要素：**

| 要素 | 内容 |
|------|------|
| 合同范围 | 设计（Engineering）、采购（Procurement）、施工（Construction）全流程 |
| 合同价款 | 固定总价（lump-sum）或成本加酬金，固定总价更为常见 |
| 工期 | 合同工期（含设计周期、采购周期、施工周期）、里程碑节点 |
| 性能指标 | 产出说明（Output Specification）、技术规范书（Technical Specification） |
| 验收标准 | 竣工验收、调试、试运行、产能达标等阶段标准 |
| 担保义务 | 履约保函、预付款保函、质保期保函 |

**合同结构通常包括：**
1. 合同协议书
2. 通用合同条件
3. 专用合同条件
4. 附件（技术规范、图纸、进度计划、变更管理程序等）

---

### 1.2 Common Dispute Clauses

**1. 变更条款（Variation/Change Orders）**
EPC合同中变更争议最为常见。典型问题包括：
- 发包人指令变更的范围界定：哪些属于合同内工作、哪些构成合同变更
- 变更价款的确定方式：采用定额计价、协商价、市场价何种标准
- 变更程序的瑕疵：未按约定流程发出的变更指令是否对承包商有约束力

```
示例争议条款表述：
"因发包人原因导致工程变更，承包人应在收到变更指令后14日内提出变更估价申请，
经发包人确认后调整合同价款。变更指令未经发包人书面确认的，承包人有权拒绝执行。"
```

**2. 工期延误与误期赔偿**
- 工期延误的责任划分：发包人原因（设计变更、付款延迟、提供现场延迟）vs承包人原因
- 不可抗力的认定与工期顺延程序
- 误期赔偿费（Liquidated Damages）的上限设定，通常为合同价的5%-10%
- 赶工费的主张条件与证明标准

**3. 性能指标与罚款条款**
- 产能/出力指标未达标的罚款计算公式
- 性能测试的程序与复测权利
- "性能测试失败"是否构成承包商根本违约
- 承包商援引"发包人提供的输入条件不符合要求"的抗辩

**4. 价款调整条款**
- 固定总价合同中的价格调整条件（如法律变更、物价异常波动）
- 清单漏项、项目特征描述不符时的计价争议
- 变更、签证、索赔的流程时效（28天索赔期限为FIDIC体系标配）

**5. 验收与移交条款**
- 初步验收（Taking Over）与最终验收（Final Acceptance）的区分
- 缺陷通知期（Defects Notification Period / Warranty Period）内的修复义务
- 移交证书（Taking-Over Certificate）的签发条件争议

**6. 争议解决条款**
- 协商前置程序是否构成诉讼/仲裁前置条件
- 专家评审（Engineer's Determination）程序的约束力
- 仲裁条款的适用法域、仲裁地、仲裁机构选择

---

### 1.3 Investor (甲方) Concerns

**投资者/发包人在EPC合同中的核心关切：**

**1. 项目交付确定性**
- 固定总价、固定工期的锁定，避免承包商以变更索赔突破总价
- 要求承包商承担"交钥匙"责任，即工程达到设计的产能/出力指标
- 关注承包商的项目管理能力与类似项目业绩

**2. 风险转移**
- 发包人希望尽可能将工程风险转移给承包商，包括设计风险、采购风险、施工风险
- 地质条件风险（通常由发包人承担，但承包商应在合理范围内进行现场勘察）
- 第三方延误风险的分担

**3. 性能保障**
- 明确的产出规范（Output Specification）而非描述性规范（Performance Specification）
- 性能测试程序的公正性与可操作性
- 性能罚款（LDs）是否具有足够的威慑力

**4. 知识产权**
- EPC合同产生的设计文件、专有技术的所有权归属
- 承包商保证其设计不侵犯第三方知识产权
- 发包人有权使用设计文件的范围与期限

**5. 资金安全**
- 预付款的保障措施（预付款保函、逐笔扣回机制）
- 承包商破产或无力履约时的合同终止与权利救济
- 承包商的分包商管理（避免分包商欠款导致工程延误）

**6. 退出机制**
- 承包商严重违约时的终止合同权
- 终止后的接管程序（step-in rights）及已完工程的计价
- 履约保函的兑付条件与程序

**7. 争议解决效率**
- 避免冗长的诉讼程序，倾向选择仲裁
- 紧急情况下的临时救济措施（如申请财产保全）

---

### 1.4 Contractor (乙方) Concerns

**承包商在EPC合同中的核心关切：**

**1. 合同范围确定性**
- 明确的工作范围边界，避免"范围蔓延"（Scope Creep）
- 发包人提供基础资料的准确性（地质报告、现有设施状况等）及责任归属
- 设备材料涨价风险的分担机制

**2. 变更与索赔权利**
- 变更指令的书面确认程序
- 工期索赔的程序与时效（28天通知义务）
- 费用索赔的项目构成（直接成本、管理费、利润、总部管理费、资金成本）

**3. 付款保障**
- 工程进度款的支付条件与时间节点
- 预付款保函的递减机制是否合理
- 竣工结算的审核期限（通常约定为28-56天）
- 承包商优先受偿权（建设工程价款优先受偿权）的行使

**4. 不可抗力与风险分担**
- 不可抗力的定义范围是否合理（是否包含流行病、政府行为等）
- 不可抗力期间的工期顺延与费用补偿
- 物价波动是否构成可调整合同价款的情形

**5. 性能测试的合理性**
- 测试条件是否由发包人合理控制（测试用燃料/原料/电网条件）
- 性能测试失败后是否有补救机会（grace period for re-testing）
- 性能罚款是否有合理上限

**6. 分包管理**
- 主要分包商（如设计院、设备制造商）的选择权
- 分包合同条款与主合同的衔接（背靠背条款的合理性）
- 劳务分包与专业分包的管理责任边界

**7. 终止与退出**
- 发包人终止合同时的已完工程款支付
- 因发包人原因终止合同时的利润损失赔偿
- 承包商主动终止合同的权利（通常限于发包人根本违约且持续不纠正）

## 2. 股权转让/收购协议

### 2.1 Overview and Core Elements

**合同性质：** 股权转让协议是转让方将其持有的公司股权/股份转让给受让方，由受让方取得公司相应股东权益的合同。收购协议（Share Purchase Agreement, SPA）是股权收购的核心法律文件，通常涉及目标公司的全部或控制性股权。

**主要法律依据：**
- 《中华人民共和国民法典》合同编、物权编
- 《中华人民共和国公司法》（2023年修订）
- 《上市公司收购管理办法》（如涉及上市公司）
- 《企业国有资产交易监督管理办法》（如涉及国有资产）

**核心要素：**

| 要素 | 内容 |
|------|------|
| 标的股权 | 具体持股比例、股权权属清晰性（是否存在质押、冻结等限制） |
| 转让价款 | 总价款、计价依据（净资产评估、收益法评估、市场法）、支付节奏 |
| 先决条件 | 审批程序（国资审批、外商投资审批、反垄断审查等）、陈述保证满足 |
| 交割 | 交割日、股权变更登记、证照印章移交 |
| 陈述与保证 | 卖方对目标公司状况的陈述保证、重大不利变化（MAC）条款 |
| 过渡期 | 过渡期义务、禁止重大处置、竞业禁止 |

**特殊类型——新能源项目公司股权转让：**
- 风电/光伏项目公司的股权转让通常涉及：
  - 项目核准/备案文件的变更
  - 发电业务许可证的变更
  - 电力送出/消纳协议的衔接
  - 土地/林地使用权的延续性
  - 补贴指标的归属（存量项目与新增项目不同处理）

---

### 2.2 Common Dispute Clauses

**1. 估值与价款调整条款**
- 交割日审计（Closing Audit）：实际净资产与预估数的差异调整
- 营运资本（Working Capital）调整机制
- 目标公司债务的披露与承担
- 股权计价基准日后的重大事件影响

```
示例价款调整机制：
"双方同意以2024年3月31日为基准日，目标公司100%股权价值为人民币1.2亿元。
交割日经审计的净资产若低于基准日净资产，超过部分从转让价款中扣除；
若高于基准日净资产，低于部分从转让价款中追加。"
```

**2. 陈述与保证（Representations & Warranties）**
- 卖方陈述保证的范围：公司的设立与存续、股权结构、财务报表、税务合规、环保合规、劳动人事、重要合同、诉讼仲裁等
- 陈述保证的"重大性"标准（Materiality Qualifier）
- 违反陈述保证的赔偿范围：直接损失、间接损失、可预见损失
- 陈述保证的披露（Disclosure）机制——卖方在协议中披露的例外事项不构成违反

**3. 重大不利变化（Material Adverse Change, MAC）条款**
- MAC的定义：何种情形构成"重大不利变化"（通常排除行业周期、宏观经济、疫情等）
- MAC条款的触发效果：受让方有权终止协议或放弃交割
- MAC的证明标准：客观举证还是主观判断

**4. 交割先决条件（Conditions Precedent）**
- 各方内部审批程序的完成时限
- 第三方同意（银行、合作伙伴、 landlords）的获取
- 监管审批（外资审批、反垄断审查、安全审查等）
- 受让方融资条件的满足

**5. 竞业禁止与非竞争**
- 卖方/原股东在一定期限、一定地域内的竞业禁止义务
- 关键人员（Key Personnel）的留任与竞业禁止
- 竞业禁止补偿金的合理性

**6. 税务 indemnity（税务赔偿）条款**
- 交割日前税务风险的归属与赔偿机制
- 税务稽查、评估的处理权限与费用分担
- 税务合规的陈述保证范围

**7. 争议解决**
- 股权纠纷的专属管辖约定（目标公司注册地法院）
- 涉外股权转让的适用法律、仲裁条款
- 少量股权争议是否适用仲裁

---

### 2.3 Investor (甲方/受让方) Concerns

**受让方在股权转让中的核心关切：**

**1. 目标公司真实状况**
- 财务数据的真实与完整性（资产、负债、收入、成本）
- 或有负债（未披露的担保、诉讼、行政处罚）的识别与防范
- 目标公司主要资产的权利完整性（土地、房产、知识产权的权属）
- 环保合规状况（尤其是新能源项目的环评批复、排放许可）

**2. 股权权属清晰性**
- 股权是否存在质押、冻结、查封等权利限制
- 其他股东的优先购买权是否已妥善处理
- 代持股权的清理与风险
- 国有股权转让的进场交易要求

**3. 估值与定价**
- 评估方法是否合理（资产基础法、收益法、市场法的适用性）
- 假设前提的合理性（如风电项目的利用小时数、上网电价）
- 溢价收购的合理性说明

**4. 交割确定性**
- 先决条件能否按时满足
- 卖方的配合义务是否充分
- 逾期交割的违约责任

**5. 过渡期风险管理**
- 过渡期内目标公司的经营管控权
- 禁止重大资产处置、对外担保、重大人事变动
- 过渡期重大事项的双重审批机制

**6. 卖方信用风险**
- 个人卖方的履约能力与资产状况
- 法人的卖方是否存在破产风险
- 卖方违约的救济手段是否充分

**7. 少数股权收购的特殊关切**
- 同控下转让与非同控下转让的税务差异
- 少数股东权利的保护机制
- 拖售权（Drag-along）、随售权（Tag-along）机制
- 股东协议的治理安排

---

### 2.4 Contractor (乙方/转让方) Concerns

**转让方在股权转让中的核心关切：**

**1. 价款回收确定性**
- 分期支付的时限与金额保障
- 尾款（大额尾款）的担保措施（托管账户、履约保函）
- 受让方破产/无力付款的风险防范

**2. 陈述保证责任**
- 陈述保证的范围是否合理（避免无限敞口）
- 重大性门槛的设定（避免微小瑕疵触发赔偿）
- 赔偿责任的时限（通常陈述保证的存续期为交割后1-3年，税务保证更长）
- 赔偿上限（通常为股权转让价款的一定比例）

**3. 过渡期义务**
- 过渡期内的经营自主权
- 商业决策的正常授权机制
- 配合尽调的义务边界

**4. 员工安置**
- 员工劳动关系延续性（劳动合同法第33、34条）
- 员工安置方案的协商
- 经济补偿金的责任归属

**5. 竞业禁止的合理性**
- 竞业禁止的地域与期限是否合理（通常不超过2-3年）
- 竞业禁止补偿金是否充分
- 违反竞业禁止的单方救济权

**6. 税务优化**
- 转让定价的税务合规性
- 地区性税收优惠的延续性
- 个人所得税的代扣代缴安排

**7. 交割后的合作**
- 重要证照/系统的移交程序
- 原管理层/核心人员的留任安排
- 过渡期支持与配合义务

---

## 3. 合作/投资协议

### 3.1 Overview and Core Elements

**合同性质：** 合作/投资协议是当事人之间为实现特定商业目的而建立的长期合作关系合同，区别于一次性交易，强调各方资源整合、利益共享、风险分担。国内新能源领域常见模式包括：合作开发协议、联合投资协议、项目公司股东协议等。

**主要法律依据：**
- 《中华人民共和国民法典》合同编（合伙合同章节、第三编第二章）
- 《中华人民共和国公司法》（公司设立、股东权利义务）
- 《中华人民共和国民法典》第967条-第978条（合伙合同）
- 《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》

**核心要素：**

| 要素 | 内容 |
|------|------|
| 合作目的 | 明确的商业目标（如共同开发某风电项目） |
| 出资/投入 | 各方的资源、资本、技术、渠道等投入及计价 |
| 权益分配 | 利润分配、亏损分担、股权比例 |
| 治理结构 | 决策机制、董事会/执行机构组成、表决权安排 |
| 退出机制 | 退出条件、退出价格、优先购买权、随售权/拖售权 |
| 贡献与分配不对等 | 资源贡献与权益比例可能不一一对应 |

**合作模式区分：**

1. **股权式合作**（设立项目公司）
   - 各方共同设立公司，协议体现为发起人协议/股东协议
   - 责任以出资额为限，风险相对隔离
   - 决策按股权比例或公司章程约定

2. **合同式合作**（不设立公司）
   - 各方以合伙形式合作，适用合伙企业法或民法典合伙合同
   - 合伙人承担无限连带责任
   - 灵活性高但风险较高

---

### 3.2 Common Dispute Clauses

**1. 出资义务与违约**
- 各方出资的时间节点与条件
- 逾期出资的违约责任（滞纳金、稀释、除名）
- 非货币出资的评估、交付与权利瑕疵担保

```
示例：
"任一方逾期出资超过30日的，应按逾期金额的万分之五/日向守约方支付违约金；
逾期超过90日的，守约方有权以1元/股的象征性价格认购违约方的认缴份额，
或要求违约方退出合作并返还已出资本金及按年化10%计算的利息。"
```

**2. 利润分配与亏损分担**
- 分配比例与出资比例不一致时的约定
- 利润分配的时点与条件（是否有可分配利润的认定争议）
- 亏损分担的底线（是否以出资额为限）
- 预分配（夹层融资/优先分配）机制的安排

**3. 治理结构与决策机制**
- 董事会席位分配与表决权
- 重大事项的多数决/一致决要求
- 一票否决权的设置与边界
- 僵局条款（Deadlock）：无法达成决议时的处理（第三方裁决、竞价出售等）

**4. 竞业禁止与关联交易**
- 关联交易的定义与披露机制
- 竞业禁止的主体、期限、地域范围
- 关联方认定的标准

**5. 知识产权归属**
- 合作期间产生的知识产权的归属
- 背景知识产权（Background IP）的使用授权
- 改进技术的后续归属

**6. 退出机制**
- 主动退出：条件、通知期限、退出价格计算
- 被动退出：违约、破产、清算情形下的退出
- 优先购买权（Right of First Refusal, ROFR）
- 随售权（Tag-along）与拖售权（Drag-along）
- 退出时的竞业限制延续

**7. 争议解决**
- 友好协商的时限
- 仲裁条款 vs 诉讼条款
- 适用法律

---

### 3.3 Investor (甲方) Concerns

**投资方在合作/投资协议中的核心关切：**

**1. 权益保障**
- 股权比例是否与话语权匹配
- 董事会席位与关键委员会代表权
- 重大事项的否决权（可转股债、清算优先权等）
- 投资者保护条款的充分性

**2. 收益预期**
- 投资回报的测算依据
- 分红政策与再投资收益安排
- 退出时的收益保障机制

**3. 控制权保护**
- 反稀释条款（Full Ratchet / Weighted Average）
- 优先认购权（Pre-emptive Right）
- 信息权与检查权
- 重大事项的共同决定权

**4. 尽职调查的局限性**
- 背景核查受限情况下的保护机制
- 陈述保证与赔偿条款
- 重大不利变化退出权

**5. 退出渠道**
- IPO条件不具备时的退出安排
- 出售给第三方的价格保障（优先购买权触发时的定价机制）
- 回购义务（Put Option）的触发条件与担保

**6. 治理参与**
- 参与日常经营的边界
- 财务监督权（审计权）
- 人事权（关键岗位的提名/批准）

**7. 投资者特殊权利（Term Sheet常见条款）**
- 董事会观察员席位
- 信息权（季度/年度财务报告）
- 优先清算权（Liquidation Preference）
- 保护性条款（Protective Provisions）

---

### 3.4 Contractor (乙方) Concerns

**被投资方/合作方在合作/投资协议中的核心关切：**

**1. 控制权维持**
- 避免投资方过度干预日常经营
- 稀释后的股权控制保障
- 创始团队/原股东的特殊表决权安排（如AB股）

**2. 经营自主权**
- 投资方干预的边界
- 日常经营的正常决策不受过度约束
- 激励计划的自主安排权

**3. 估值合理性**
- 估值基础是否合理
- 投资方要求的优先清算权是否过重
- 反稀释条款的影响

**4. 退出灵活性**
- 主动上市的时机选择权
- 后续融资时的拖售权/优先购买权限制
- 竞业禁止义务的合理边界

**5. 利益平衡**
- 优先股与普通股的权益差异是否合理
- 投资者回报与经营层激励的平衡
- 信息披露义务的范围

**6. 退出条款的公平性**
- 回购义务（Put Option）的对价是否合理
- 拖售权触发条件是否过于宽松
- 优先购买权的定价机制

**7. 原团队保护**
- 竞业禁止补偿是否到位
- 知识产权归属是否公平
- 离职时的股权成熟（Vesting）安排

## 4. 供货/采购合同

### 4.1 Overview and Core Elements

**合同性质：** 供货/采购合同是出卖人向买受人转移货物所有权，买受人支付价款的典型双务合同。国内新能源领域的设备采购合同通常具有标的大、技术要求高、交货周期长、售后服务要求高等特点。

**主要法律依据：**
- 《中华人民共和国民法典》合同编（第三编第五章——买卖合同）
- 《中华人民共和国产品质量法》
- 《中华人民共和国标准化法》
- 《最高人民法院关于审理买卖合同纠纷案件适用法律问题的解释》（法释〔2020〕17号）

**核心要素：**

| 要素 | 内容 |
|------|------|
| 标的物 | 设备/货物名称、规格型号、数量、技术参数 |
| 质量标准 | 国家标准、行业标准、企业标准、或技术协议约定 |
| 价款 | 单价、总价、定价机制（固定价、市场价、成本加酬金） |
| 交货 | 交货时间、地点、运输方式、风险转移（Incoterms 2020） |
| 验收 | 初步验收、终验、质量异议期 |
| 质保 | 质保期、质保条件、质保责任 |
| 付款 | 付款节点（预付款、到货款、验收款、质保金）、付款条件 |

**Incoterms 2020 常用条款（进出口场景）：**
- EXW（Ex Works）：卖方工厂交货，风险最小
- FOB（Free on Board）：装运港船上交货
- CIF（Cost, Insurance and Freight）：成本+保险+运费
- DAP（Delivered at Place）：目的地交货
- DDP（Delivered Duty Paid）：完税后交货

---

### 4.2 Common Dispute Clauses

**1. 质量异议条款**
- 质量异议期的设定（通常为收货后30-90天内提出书面异议）
- 质量异议的处理程序：检验、通知、复检
- 质量异议期届满的法律效果（视为验收合格，但隐蔽瑕疵除外）
- 质量不符合合同约定的违约金计算（通常约定为问题批次货值的5%-20%）

```
示例质量异议条款：
"买方应在收到货物后30日内完成数量清点及外观检验，并在90日内完成安装调试检验。
买方发现货物存在质量问题或规格不符的，应在发现缺陷后15日内书面通知卖方，
并提供第三方检验报告。卖方应在收到异议通知后30日内派人核实并提出处理方案。"
```

**2. 付款与发票**
- 付款节点与验收确认的关系
- 背靠背付款条款（等上游方付款后再付下游方）
- 发票的类型（增值税专用发票）与开具时间
- 逾期付款的违约金（通常为日万分之五至万分之三）

**3. 交货与运输**
- 交货时间的约束力（迟延交货违约金）
- 不可抗力导致交货延迟的处理
- 运输途中的风险转移（买方或卖方承担）
- 分批交货的安排与违约责任

**4. 所有权保留（Retention of Title）**
- 民法典第641条：约定买方付清全款前所有权保留
- 保留所有权的登记（对抗善意第三人）
- 分期付款中保留所有权条款的效力

**5. 知识产权**
- 卖方保证其提供的货物不侵犯第三方知识产权
- 侵权时的赔偿责任与抗辩
- 授权使用的范围

**6. 售后服务与质保**
- 质保期的起算点（交货日、验收日、安装调试完成日）
- 质保期内的服务内容（免费维修、零部件更换）
- 质保期外的有偿服务安排
- 售后服务响应时限

**7. 退货/换货/降价处理**
- 质量问题严重程度与处理方式的对应关系
- 换货后的质保期是否重新起算
- 降价处理的计价方式

**8. 争议解决**
- 货物质量争议的检验/鉴定机构选择
- 仲裁条款的效率优势

---

### 4.3 Investor (甲方/买方) Concerns

**买方在供货/采购合同中的核心关切：**

**1. 质量保障**
- 明确的技术规范书（Technical Specification）减少争议
- 质量保证体系的证明（ISO认证等）
- 出厂检验（FAT）和现场检验（SAT）程序
- 质量问题导致项目延误的责任归属

**2. 供货确定性**
- 供应商的生产能力与产能储备
- 关键原材料的供应链稳定性
- 替代供应商的储备（避免单一来源风险）
- 供应商破产/违约时的应对措施

**3. 交付保障**
- 明确的交货时间表与里程碑
- 迟延交货的违约责任（误期赔偿）
- 分批交货的衔接安排
- 运输保险的覆盖范围

**4. 价格锁定**
- 固定价的锁定期
- 原材料价格波动时的调价机制（公式调价）
- 汇率风险的承担
- 订单最小量/年度最低采购量的约定

**5. 技术支持**
- 安装指导、现场调试服务
- 操作培训的内容与时长
- 远程诊断与技术支持能力
- 技术资料的交付（图纸、操作手册、维修手册等）

**6. 备件供应**
- 初始备件清单与价格
- 质保期后备件的长期供应承诺
- 备件价格与供货时限

**7. 验收主导权**
- 验收标准的明确性
- 验收程序的公正性
- 验收不合格时的处理机制（修复/换货/降价/退货）

---

### 4.4 Contractor (乙方/卖方) Concerns

**卖方在供货/采购合同中的核心关切：**

**1. 付款确定性**
- 预付款的安全性（预付款保函或信用证）
- 付款与交货的节点匹配
- 买方破产/无力付款的风险
- 长期质保金的释放保障

**2. 合同范围确定性**
- 规格型号、技术方案的确认程序
- 避免"技术变更"导致成本增加
- 买方提供基础条件的责任边界

**3. 原材料价格风险**
- 长周期供货合同中原材料涨价风险
- 调价公式的公平性
- 最低订单量与备货义务

**4. 质量责任边界**
- 质保期的合理长度（通常为验收后12-24个月）
- 质保责任的构成要件（正常使用、无人为损坏）
- 质保责任与产品责任的区分
- 质保金（通常为合同价的5%-10%，质保期满后释放）

**5. 出口合规**
- 出口管制与制裁合规
- 两用物项的出口许可
- 海关编码与监管条件

**6. 知识产权保护**
- 卖方专有技术的保护
- 买方使用技术资料的范围限制
- 仿制/逆向工程的禁止

**7. 竞业限制**
- 买方不得将卖方技术用于竞争目的
- 备件供应的独家性保障

---

## 5. 融资租赁合同

### 5.1 Overview and Core Elements

**合同性质：** 融资租赁合同是出租人根据承租人对租赁物和出卖人的选择，向出卖人购买租赁物，提供给承租人使用，承租人支付租金的合同。民法典第十五章（第735条-第752条）对融资租赁合同作了专章规定，确立了融资租赁的有名合同地位。

**主要法律依据：**
- 《中华人民共和国民法典》第十五章（第735-752条）
- 《最高人民法院关于审理融资租赁合同纠纷案件适用法律问题的解释》（法释〔2020〕17号）
- 《金融租赁公司管理办法》（银保监会令〔2014〕第3号）
- 《融资租赁企业监督管理办法》（商服贸函〔2013〕第337号）

**核心要素：**

| 要素 | 内容 |
|------|------|
| 租赁物 | 设备/资产的具体描述、权属状况、价值评估 |
| 租金 | 租金总额、租期、每期租金、租金构成（本金+利息） |
| 租赁成本 | 租赁物的购买价款（设备原价、安装费、运费等） |
| 租期 | 租赁期限（通常与设备折旧年限匹配） |
| 所有权 | 租赁期间出租人对租赁物的所有权 |
| 期满选择权 | 留购、续租、退租的选择及对价 |

**融资租赁 vs 经营租赁的核心区别：**
- 融资租赁：实质上转移了与租赁物所有权有关的全部风险和报酬，租期结束时承租人通常取得所有权
- 经营租赁：出租人承担租赁物所有权的主要风险，承租人仅为使用

**租金计算逻辑（等额本金法——新能源项目常用）：**
```
每期租金 = 租赁本金/租期 +当期租赁本金余额 × 年利率/12
租赁本金余额随每期还款递减
```

---

### 5.2 Common Dispute Clauses

**1. 租赁物的瑕疵担保责任（民法典第747条）**
- 租赁物不符合约定或使用目的的，承租人有权要求出租人承担责任
- 出租人不承担瑕疵担保责任的例外：出租人干预选择、明知瑕疵不告知
- 承租人不得以租赁物瑕疵为由拒付或迟延支付租金

```
示例：
"租赁物存在质量瑕疵的，承租人应直接向出卖人主张权利，出租人应当协助承租人行使索赔权。
承租人已直接向出卖人提出索赔的，不影响其按照本合同的约定支付租金。"
```

**2. 租金支付与加速到期**
- 租金支付的时点与账户
- 逾期付款的违约金（日万分之五至万分之三）
- 加速到期条款：承租人违约时，出租人有权宣布全部未到期租金立即到期
- 出租人终止合同、收回租赁物的条件

**3. 租赁物所有权与登记（民法典第745条）**
- 融资租赁登记的效力（对抗善意第三人）
- 租赁物所有权的证明（采购合同、发票、付款凭证）
- 租赁物上是否存在抵押/质押（出租人应确保租赁物无权利负担）
- 承租人破产时租赁物的取回权

**4. 租赁物的使用、保管与维修**
- 承租人对租赁物的妥善保管义务
- 维修义务的归属（通常由承租人承担）
- 租赁物损坏/灭失的风险承担
- 承租人破产时租赁物的取回

**5. 租赁物的改善与添附**
- 承租人未经出租人同意不得对租赁物进行改善或增设他物
- 改善或增设他物的归属与补偿
- 租赁物上新增设施的处理

**6. 转让与转租**
- 承租人转让租赁物或转租的限制
- 承租人破产时租赁物是否纳入破产财产
- 出租人转让融资租赁合同权利义务

**7. 期满选择权**
- 留购价格的设定（通常为象征性价格如1元或公允价值）
- 续租的租金调整机制
- 退租时租赁物的状态要求（正常磨损除外）

**8. 争议解决**
- 融资租赁纠纷的诉讼/仲裁管辖
- 担保物权实现（对保证人、抵押物的执行）

---

### 5.3 Investor (甲方/出租人) Concerns

**出租人在融资租赁合同中的核心关切：**

**1. 租金安全**
- 租金支付的确定性（与承租人信用挂钩）
- 承租人违约时的加速到期权
- 租赁物残值保障（避免租期结束时租赁物大幅贬值）
- 担保措施：保证、抵押、质押、租金保理

**2. 所有权保障**
- 融资租赁登记的办理（对抗善意取得）
- 租赁物不被非法处置、抵押、转让
- 承租人破产时租赁物的取回权行使
- 租赁物的保险安排（财产一切险/机器损坏险）

**3. 税务优化**
- 增值税抵扣：融资租赁增值税发票（13%税率可抵扣）
- 企业所得税：租赁资产计提折旧的税务处理
- 租赁物折旧年限与租期的匹配

**4. 租赁物价值保障**
- 租赁物的选择标准（通用性、可变现性）
- 二手市场的活跃程度
- 关键设备的厂商回购承诺（OEM Buyback）

**5. 监管合规**
- 金融租赁公司需满足资本充足率、资产规模等监管要求
- 融资租赁公司需符合商务部门监管要求
- 跨境融资租赁的外汇管理要求

**6. 违约处理**
- 收回租赁物的程序与成本
- 租赁物的处置渠道
- 损失追偿的路径（承租人、其他担保人）

**7. 承租人信用风险**
- 承租人的财务状况与偿付能力
- 关联公司的保证担保
- 承租人实际控制人的连带责任

---

### 5.4 Contractor (乙方/承租人) Concerns

**承租人在融资租赁合同中的核心关切：**

**1. 租金成本**
- 租金水平是否具有市场竞争力
- 利率是固定还是浮动（利率上浮的风险承担）
- 提前还款的罚息/违约金
- 租金支付的节奏安排

**2. 租赁物的选择与质量**
- 自主选择租赁物和出卖人的权利（民法典第735条）
- 出卖人不履行义务时向出卖人直接主张的权利
- 租赁物瑕疵不影响租金支付义务的例外

**3. 所有权归属**
- 期满后取得所有权的确定性（留购价格的合理性）
- 租期结束时续租或退租的选择权
- 租赁物过户的税费承担

**4. 使用自主权**
- 租赁物能否用于抵押/质押（通常受限）
- 租赁物的转租限制
- 租赁物迁移的限制

**5. 税务处理**
- 租金的增值税进项抵扣
- 租赁资产折旧的税前扣除
- 融租租赁的会计处理（融资租入固定资产入账）

**6. 提前终止**
- 提前终止合同的权利
- 提前终止时的补偿计算（剩余租期全部租金+合理利润）
- 违约金的合理性上限

**7. 争议解决效率**
- 诉讼/仲裁的成本与效率
- 资产取回程序的复杂性
